# よく作成される機能一覧

## company-core（MU Plugin）によく作る機能

サイトの骨格となる機能。消えたらサイトが壊れるものをここに置く。

| 機能 | 概要 |
|---|---|
| カスタム投稿タイプ | お知らせ・イベント・求人・物件など |
| カスタムタクソノミー | カテゴリ・タグの独自分類 |
| ACFフィールド定義 | 入力項目をコードで管理 |
| 管理画面カスタマイズ | 不要メニュー非表示・ダッシュボード変更 |
| ユーザーロール設定 | 権限グループの追加・変更 |
| メール送信処理 | お問い合わせ・通知メールのロジック |
| Slack通知 | 投稿公開・フォーム送信時の通知 |
| 外部API連携基盤 | HubSpot・kintone・Salesforceなど |
| REST API拡張 | 独自エンドポイントの追加 |
| リライトルール | 独自URLパターンの追加 |

---

## my-custom-plugin によく作る機能

独立した機能単位。無効化しても他に影響しないものをここに置く。

| 機能 | 概要 |
|---|---|
| 会員機能 | 登録・ログイン・マイページ |
| 会員ランク判定 | ゴールド・シルバーなどのランク管理 |
| 予約管理 | カレンダー・空き確認・予約フォーム |
| CSV入出力 | 投稿データの一括インポート・エクスポート |
| バッチ処理 | WP-Cronを使った定期実行処理 |
| 検索カスタマイズ | 絞り込み・ソート・全文検索 |
| WooCommerce拡張 | 独自送料・決済・購入フロー変更 |
| LINE連携 | LINE Messaging APIを使った通知・チャット |
| マップ表示 | Google Maps API連携 |
| お気に入り機能 | 投稿・物件などのブックマーク |

---

## どちらに置くか判断基準

| 判断軸 | 置き場所 |
|---|---|
| 消えたらサイトが壊れる | company-core（MU Plugin） |
| 投稿タイプ・タクソノミー定義 | company-core（MU Plugin） |
| 無効化しても他に影響ない | my-custom-plugin |
| 機能単位で切り出せる | my-custom-plugin |
